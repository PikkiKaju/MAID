# Dataset Components

Komponenty do zarządzania zbiorami danych w panelu administracyjnym.

## Komponenty

- **DatasetList.tsx** - Lista wszystkich zbiorów danych z możliwością filtrowania i sortowania
- **DatasetEdit.tsx** - Formularz edycji zbioru danych z wyświetlaniem szczegółów
- **DatasetFilter.tsx** - Komponent filtrowania zbiorów danych (po statusie, właścicielu, dacie)
- **index.ts** - Eksport wszystkich komponentów

## Funkcjonalność

Komponenty umożliwiają:

- Przeglądanie listy wszystkich zbiorów danych w systemie
- Filtrowanie zbiorów danych po różnych kryteriach
- Edycję szczegółów zbioru danych (nazwa, opis, status publiczny/prywatny)
- Wyświetlanie informacji o właścicielu zbioru danych
- Zarządzanie widocznością zbiorów danych

## Technologie

Komponenty wykorzystują React Admin do zarządzania danymi i Material-UI do stylizacji.
