# Project Components

Komponenty do zarządzania projektami w panelu administracyjnym.

## Komponenty

- **ProjectList.tsx** - Lista wszystkich projektów z możliwością filtrowania i sortowania
- **ProjectEdit.tsx** - Formularz edycji projektu z wyświetlaniem szczegółów
- **ProjectFilter.tsx** - Komponent filtrowania projektów (po statusie, właścicielu, dacie)
- **index.ts** - Eksport wszystkich komponentów

## Funkcjonalność

Komponenty umożliwiają:

- Przeglądanie listy wszystkich projektów w systemie
- Filtrowanie projektów po różnych kryteriach
- Edycję szczegółów projektu (nazwa, opis, status, widoczność)
- Wyświetlanie informacji o właścicielu projektu
- Zarządzanie widocznością projektów (publiczny/prywatny)

## Technologie

Komponenty wykorzystują React Admin do zarządzania danymi i Material-UI do stylizacji.
