# Layout Components

Komponenty układu aplikacji panelu administracyjnego.

## Komponenty

- **CustomLayout.tsx** - Główny komponent układu aplikacji z niestandardowym tłem gradientowym
- **CustomAppBar.tsx** - Niestandardowy pasek aplikacji z menu użytkownika i opcją dodawania administratorów
- **CustomLoginPage.tsx** - Niestandardowa strona logowania
- **AddAdminDialog.tsx** - Dialog do dodawania nowego administratora

## Funkcjonalność

Komponenty zapewniają:

- Spójny układ aplikacji z niestandardowym designem
- Pasek aplikacji z tytułem i menu użytkownika
- Możliwość dodawania nowych administratorów bezpośrednio z menu
- Niestandardową stronę logowania
- Obsługę motywu jasnego i ciemnego

## Integracja

Komponenty są zintegrowane z React Admin i wykorzystują Material-UI do stylizacji.
