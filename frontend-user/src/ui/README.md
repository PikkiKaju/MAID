# UI Components

Ten folder zawiera podstawowe komponenty UI zbudowane na bazie Radix UI i shadcn/ui.

## Komponenty

Wszystkie komponenty są zbudowane zgodnie z wzorcem shadcn/ui i używają Tailwind CSS do stylowania.

- **alert-dialog.tsx** - Dialog z potwierdzeniem akcji
- **avatar.tsx** - Komponent avatara użytkownika
- **badge.tsx** - Badge/znacznik
- **button.tsx** - Przycisk z różnymi wariantami
- **card.tsx** - Karta/kontener
- **dialog.tsx** - Modal dialog
- **dropdown-menu.tsx** - Menu rozwijane
- **input.tsx** - Pole tekstowe
- **label.tsx** - Etykieta dla inputów
- **progress.tsx** - Pasek postępu
- **select.tsx** - Lista rozwijana
- **separator.tsx** - Separator wizualny
- **sheet.tsx** - Panel boczny
- **sidebar.tsx** - Komponenty sidebar
- **skeleton.tsx** - Placeholder podczas ładowania
- **switch.tsx** - Przełącznik on/off
- **tabs.tsx** - Zakładki
- **textarea.tsx** - Wieloliniowe pole tekstowe
- **tooltip.tsx** - Tooltip/podpowiedź

## Użycie

Wszystkie komponenty są dostępne w całej aplikacji i używane w komponentach biznesowych.
