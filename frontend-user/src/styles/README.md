# Styles

Ten folder zawiera globalne style CSS aplikacji.

## Pliki

- **globals.css** - Globalne style CSS z definicjami zmiennych, resetem i podstawowymi stylami
- **Loader.css** - Style dla komponentu loader/spinner

## Użycie

Style są importowane w `main.tsx` lub `App.tsx` i dostępne globalnie w całej aplikacji.
