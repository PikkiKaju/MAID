# Internationalization (i18n)

Ten folder zawiera konfigurację internacjonalizacji aplikacji.

## Pliki

- **config.ts** - Konfiguracja i18next z ustawieniami języków (polski, angielski) i źródłami tłumaczeń

## Użycie

i18n jest inicjalizowany w `main.tsx` i używany w komponentach przez hook `useTranslation()`.

## Języki

- Polski (pl) - domyślny
- Angielski (en)

Tłumaczenia znajdują się w folderze `/messages`.
