# Header Components

Komponenty związane z nagłówkiem aplikacji.

## Komponenty

- **ProfileDropdown.tsx** - Menu rozwijane z profilem użytkownika, zawierające opcje:
  - Wyświetlanie awatara i nazwy użytkownika
  - Link do profilu
  - Wylogowanie

## Funkcjonalność

Komponent umożliwia:

- Wyświetlanie informacji o zalogowanym użytkowniku
- Nawigację do profilu użytkownika
- Wylogowanie z aplikacji
- Integrację z Redux do zarządzania stanem autoryzacji
