# Form Components

Komponenty formularzy autoryzacji.

## Komponenty

- **LoginForm.tsx** - Formularz logowania z polami: username, password
- **RegisterForm.tsx** - Formularz rejestracji z polami: username, email, password, confirmPassword

## Funkcjonalność

Formularze:

- Walidują dane po stronie klienta
- Wyświetlają błędy z backendu
- Obsługują tłumaczenia (i18n)
- Integrują się z Redux do zarządzania stanem autoryzacji
