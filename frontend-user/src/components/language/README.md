# Language Components

Komponenty do zarządzania językiem aplikacji.

## Komponenty

- **LanguageSwitcher.tsx** - Przełącznik języka (polski/angielski)

## Funkcjonalność

Komponent umożliwia:

- Przełączanie między językami aplikacji
- Wyświetlanie aktualnego języka
- Zapisywanie preferencji użytkownika
