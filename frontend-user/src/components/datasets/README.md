# Dataset Components

Komponenty do zarządzania zbiorami danych.

## Komponenty

- **UploadArea.tsx** - Obszar do przeciągania i upuszczania plików
- **UploadDatasetDialog.tsx** - Dialog do przesyłania nowego zbioru danych
- **DatasetDetailsDialog.tsx** - Dialog ze szczegółami zbioru danych
- **AttachedDatasets.tsx** - Lista załączonych zbiorów danych do projektu
- **PublicDatasetsSection.tsx** - Sekcja z publicznymi zbiorami danych
- **HeaderDatasets.tsx** - Nagłówek sekcji zbiorów danych
- **Tips.tsx** - Wskazówki dotyczące zbiorów danych

## Funkcjonalność

Komponenty umożliwiają:

- Przesyłanie plików CSV i zdjęć
- Przeglądanie listy zbiorów danych (własnych i publicznych)
- Wyświetlanie szczegółów zbioru danych
- Zarządzanie załączonymi zbiorami danych w projekcie
