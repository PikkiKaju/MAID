# Home Page Components

Komponenty strony głównej aplikacji.

## Komponenty

- **RecentSection.tsx** - Sekcja z ostatnio oglądanymi projektami
- **TrendingSection.tsx** - Sekcja z popularnymi projektami
- **FavoritesSection.tsx** - Sekcja z ulubionymi projektami
- **CategorySection.tsx** - Sekcja z kategoriami projektów
- **CategoryGrid.tsx** - Siatka kategorii
- **SearchResultsSection.tsx** - Sekcja z wynikami wyszukiwania

## Funkcjonalność

Komponenty wyświetlają różne sekcje na stronie głównej, umożliwiając użytkownikom szybki dostęp do treści.
