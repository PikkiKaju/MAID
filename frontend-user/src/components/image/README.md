# Image Components

Komponenty do obsługi obrazów.

## Komponenty

- **ImageWithFallback.tsx** - Komponent obrazu z fallbackiem w przypadku błędu ładowania

## Funkcjonalność

Komponent zapewnia:

- Wyświetlanie obrazu z podanego URL
- Automatyczne przełączenie na obraz zastępczy w przypadku błędu
- Obsługę stanu ładowania
