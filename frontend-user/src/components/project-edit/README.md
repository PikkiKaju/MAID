# Project Edit Components

Komponenty do edycji i konfiguracji projektów.

## Komponenty

- **ProjectEditSidebar.tsx** - Boczny panel z formularzem edycji projektu, zawierający wszystkie pola konfiguracyjne
- **ProjectEditTopbar.tsx** - Górny pasek narzędzi w edytorze projektu z przyciskami akcji
- **AlgorithmSelect.tsx** - Komponent do wyboru algorytmu uczenia maszynowego
- **DatasetSelect.tsx** - Komponent do wyboru zbioru danych dla projektu
- **ColumnSelect.tsx** - Komponent do wyboru kolumn z wybranego zbioru danych
- **ParameterInputs.tsx** - Komponent do wprowadzania parametrów algorytmu (np. learning rate, epochs)
- **StatusSelect.tsx** - Komponent do wyboru statusu projektu
- **PublicCheckbox.tsx** - Checkbox do ustawienia projektu jako publiczny/prywatny
- **CalculationResults.tsx** - Komponent do wyświetlania wyników obliczeń i metryk

## Funkcjonalność

Komponenty umożliwiają:

- Edycję szczegółów projektu (nazwa, opis, status)
- Konfigurację algorytmu uczenia maszynowego
- Wybór i konfigurację zbioru danych
- Ustawienie parametrów treningu
- Zarządzanie widocznością projektu (publiczny/prywatny)
- Wyświetlanie wyników obliczeń i metryk modelu
