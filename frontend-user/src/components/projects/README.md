# Project Components

Komponenty do zarządzania projektami.

## Komponenty

- **ProjectsGrid.tsx** - Siatka z projektami użytkownika
- **ProjectCard.tsx** - Karta pojedynczego projektu z podstawowymi informacjami
- **CreateProjectWindow.tsx** - Okno do tworzenia nowego projektu
- **DeleteConfirmationDialog.tsx** - Dialog potwierdzenia usunięcia projektu
- **FiltersAndSearch.tsx** - Komponenty filtrowania i wyszukiwania projektów
- **EmptyState.tsx** - Komponent wyświetlany gdy brak projektów
- **HeaderProfile.tsx** - Nagłówek sekcji projektów

## Funkcjonalność

Komponenty umożliwiają:

- Przeglądanie listy projektów
- Tworzenie nowych projektów
- Edycję i usuwanie projektów
- Filtrowanie i wyszukiwanie projektów
- Wyświetlanie stanu pustego (gdy brak projektów)
