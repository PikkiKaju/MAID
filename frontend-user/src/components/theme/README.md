# Theme Components

Komponenty do zarządzania motywem aplikacji (jasny/ciemny).

## Komponenty

- **ThemeProvider.tsx** - Provider kontekstu motywu, zarządza stanem motywu i synchronizuje z localStorage
- **ThemeToggle.tsx** - Przycisk/przełącznik do zmiany motywu (jasny/ciemny/system)

## Funkcjonalność

Komponenty umożliwiają:

- Przełączanie między motywem jasnym, ciemnym i systemowym
- Zapisywanie preferencji użytkownika w localStorage
- Automatyczne zastosowanie motywu przy załadowaniu aplikacji
