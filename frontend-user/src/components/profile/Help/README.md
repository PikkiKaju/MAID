# Profile Help Components

Komponenty sekcji pomocy w profilu użytkownika.

## Komponenty

- **FAQList.tsx** - Lista często zadawanych pytań z możliwością filtrowania
- **QuickActions.tsx** - Szybkie akcje pomocy (linki, przyciski)
- **SearchBar.tsx** - Wyszukiwarka w sekcji FAQ
- **ContactAndResources.tsx** - Informacje kontaktowe i zasoby pomocowe

## Funkcjonalność

Komponenty umożliwiają użytkownikom:

- Przeglądanie FAQ
- Wyszukiwanie odpowiedzi na pytania
- Szybki dostęp do zasobów pomocy
- Kontakt z supportem
