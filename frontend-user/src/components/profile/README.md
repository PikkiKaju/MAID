# Profile Components

Komponenty związane z profilem użytkownika.

## Komponenty

- **ProfileOverview.tsx** - Przegląd profilu z avatarem, imieniem, tytułem, bio i datą dołączenia
- **PersonalInfoForm.tsx** - Formularz edycji danych osobowych (imię, nazwisko, email, tytuł, bio, avatar)
- **SecurityForm.tsx** - Formularz zmiany hasła
- **DangerZoneCard.tsx** - Karta z opcją usunięcia konta
- **AvatarPicker.tsx** - Dialog do wyboru avatara z listy dostępnych
- **StatsGrid.tsx** - Siatka ze statystykami (projekty, zbiory danych)
- **AchievementsGrid.tsx** - Siatka z osiągnięciami użytkownika
- **SkillsList.tsx** - Lista umiejętności
- **RecentActivity.tsx** - Lista ostatniej aktywności
- **NotificationsCard.tsx** - Karta z powiadomieniami

### `/Help`

Komponenty pomocy i wsparcia:

- **FAQList.tsx** - Lista często zadawanych pytań
- **QuickActions.tsx** - Szybkie akcje pomocy
- **SearchBar.tsx** - Wyszukiwarka w FAQ
- **ContactAndResources.tsx** - Kontakt i zasoby

## Funkcjonalność

Komponenty umożliwiają:

- Wyświetlanie informacji o profilu
- Edycję danych osobowych
- Zmianę hasła
- Wybór avatara
- Usuwanie konta
- Przeglądanie statystyk i aktywności
