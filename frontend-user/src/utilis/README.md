# Utilities

Ten folder zawiera funkcje pomocnicze i narzędzia używane w całej aplikacji.

## Pliki

- **functions.ts** - Różne funkcje pomocnicze (np. `capitalizeFirstLetter`)
- **tailwind.ts** - Funkcje pomocnicze dla Tailwind CSS (np. `cn` do łączenia klas)
- **fileUpload.ts** - Funkcje do obsługi uploadu plików
- **drag-and-drop.ts** - Funkcje do obsługi drag & drop

## Użycie

Funkcje są importowane i używane w komponentach, serwisach i innych częściach aplikacji.
