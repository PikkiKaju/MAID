# Search Feature

Feature Redux do zarządzania wyszukiwaniem.

## Pliki

- **searchSlice.ts** - Redux slice ze stanem wyszukiwania:
  - `term` - Wyszukiwany termin
  - `results` - Wyniki wyszukiwania

## Funkcjonalność

Feature zarządza:

- Terminem wyszukiwania wprowadzonym przez użytkownika
- Wynikami wyszukiwania
- Synchronizacją stanu wyszukiwania w całej aplikacji
