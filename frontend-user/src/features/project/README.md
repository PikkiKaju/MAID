# Project Feature

Feature Redux do zarządzania projektami.

## Pliki

- **projectSlice.ts** - Redux slice ze stanem projektów
- **projectThunks.ts** - Async thunks do operacji na projektach:
  - Tworzenie projektu
  - Pobieranie listy projektów
  - Pobieranie szczegółów projektu
  - Aktualizacja projektu
  - Usuwanie projektu

## Funkcjonalność

Feature zarządza:

- Stanem projektów użytkownika
- Operacjami CRUD na projektach
- Synchronizacją z backendem API
