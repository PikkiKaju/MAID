# Custom Hooks

Ten folder zawiera niestandardowe hooki React używane w aplikacji.

## Hooks

- **useMobile.ts** - Hook do wykrywania urządzeń mobilnych
- **useProjects.ts** - Hook do zarządzania projektami (pobieranie, filtrowanie, sortowanie)

## Użycie

Hooki są używane w komponentach do enkapsulacji logiki i ponownego użycia.
