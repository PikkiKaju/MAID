# Pages

Ten folder zawiera główne strony/komponenty stron aplikacji.

## Strony

- **AppLayout.tsx** - Główny layout aplikacji z Header, Sidebar i Outlet dla routingu
- **HomePage.tsx** - Strona główna z sekcjami: Recent, Trending, Favorites, Categories
- **LoginPage.tsx** - Strona logowania
- **RegisterPage.tsx** - Strona rejestracji
- **ProjectsPage.tsx** - Strona z listą projektów użytkownika
- **ProjectEditPage.tsx** - Strona edycji projektu
- **DatasetsListPage.tsx** - Strona z listą zbiorów danych
- **CanvasPage.tsx** - Strona do wizualizacji i edycji modeli sieci neuronowych
- **ProfileInfoPage.tsx** - Strona z informacjami o profilu użytkownika
- **ProfileSettingPage.tsx** - Strona ustawień profilu (edycja danych, zmiana hasła, usuwanie konta)
- **ProfileHelpPage.tsx** - Strona pomocy i wsparcia
- **NotFoundPage.tsx** - Strona 404

## Routing

Wszystkie strony są zdefiniowane w `/router/router.tsx`.
